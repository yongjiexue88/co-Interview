Sliding Triplet Sum (Array Manipulation)</h3><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Problem (CodeSignal style):</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Given an array a, return an array b of the same length where for each i: b[i] = a[i - 1] + a[i] + a[i + 1] If an index is out of bounds, use 0 for that term.</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Input/Output example:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">For a = [4, 0, 1, -2, 3], solution(a) = [4, 5, -1, 2, 1].</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Python solution (adaptable to other languages):</strong><br><br>def solution(a):<br>n = len(a)<br>b = [0] * n</p><p class="mb-6 text-gray-300 leading-relaxed">for i in range(n):<br>s = a[i]</p><p class="mb-6 text-gray-300 leading-relaxed">if i &gt; 0:<br>s += a[i - 1]</p><p class="mb-6 text-gray-300 leading-relaxed">if i &lt; n - 1:<br>s += a[i + 1]</p><p class="mb-6 text-gray-300 leading-relaxed">b[i] = s</p><p class="mb-6 text-gray-300 leading-relaxed">return b<br><br><strong class="font-bold text-white">Reasoning, step by step:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Initialize b with zeros to hold results for every index.</li><li class="text-gray-300">For each index i, start with a[i]. That trades a small amount of memory for simpler indexing.</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Skills tested:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Array traversal, boundary handling, fundamental indexing, and careful edge case handling.</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Complexity:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300"><strong class="font-bold text-white">Time:</strong> O(n) where n = len(a). Vowel Consonant Pattern Match (String Pattern Matching)</h3><p class="mb-6 text-gray-300 leading-relaxed">Problem (CodeSignal style):</p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Given pattern (string of '0' and '1') and source (lowercase letters), count substrings of source that match pattern length where:<ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">'0' in pattern means vowel in substring</li><li class="text-gray-300">1' means consonant in substringVowels set = {a, e, i, o, u, y}.</li></ul></li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Input/Output examples:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">pattern = "010", source = "amazing" → solution(pattern, source) = 2</li><li class="text-gray-300">pattern = "100", source = "codesignal" → solution(pattern, source) = 0</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Python solution:</strong><br><br>VOWELS = set('aeiouy')</p><p class="mb-6 text-gray-300 leading-relaxed">def matches_at(pattern, source, start):<br>"""Check if pattern matches source at given start index."""<br>for k, ch in enumerate(pattern):<br>c = source[start + k]</p><p class="mb-6 text-gray-300 leading-relaxed">if pattern[k] == '0':<br>if c not in VOWELS:<br>return False<br>else:<br>if c in VOWELS:<br>return False</p><p class="mb-6 text-gray-300 leading-relaxed">return True</p><p class="mb-6 text-gray-300 leading-relaxed">def solution(pattern, source):<br>"""Count occurrences of pattern in source."""<br>lp = len(pattern)<br>ls = len(source)</p><p class="mb-6 text-gray-300 leading-relaxed">if lp &gt; Pair Sums That Are Powers of Two (Lookup Table)</h3><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Problem (CodeSignal style):</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Given an array of unique integer numbers, count pairs (i, j) with i ≤ j such that numbers[i] + numbers[j] equals a power of two (1, 2, 4, 8, ...).</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Input/Output examples:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">numbers = [1, -1, 2, 3] → solution(numbers) = 5</li><li class="text-gray-300">numbers = [2] → solution(numbers) = 1</li><li class="text-gray-300">numbers = [-2, -1, 0, 1, 2] → solution(numbers) = 5</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Guaranteed constraints:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">1 ≤ n ≤ 10^5, numbers[i] in [-10^6, 10^6]</li></ul><p class="mb-6 text-gray-300 leading-relaxed"><strong class="font-bold text-white">Python solution:</strong><br><br>from collections import defaultdict</p><p class="mb-6 text-gray-300 leading-relaxed">def solution(numbers):<br>counts = defaultdict(int)<br>answer = 0</p><p class="mb-6 text-gray-300 leading-relaxed"># Precompute powers of two (up to 2^21, which covers numbers up to ~2e6)<br>powers = [1 &lt;&lt; k for k in range(22)]</p><p class="mb-6 text-gray-300 leading-relaxed">for x in numbers:<br># For each power, look for complement already seen<br>for p in powers:<br>need = p - x<br>answer += counts.get(need, 0)<br>counts[x] += 1</p><p class="mb-6 text-gray-300 leading-relaxed">return answer<br><br><br><strong class="font-bold text-white">Reasoning, step by step:</strong></p><ul class="mb-6 space-y-2 pl-6"><li class="text-gray-300">Use a dictionary to count how many times a value has appeared while scanning left to right.</li><li class="text-gray-300">For the current element x, for each relevant power of two p, compute need = p - x. If a need was seen before, each occurrence forms a valid pair with the current x and increases the count by counts[need].</li><li class="text-gray-300">After checking all powers, record x in counts so later elements can pair with it.</li><li class="text-gray-300">The range of powers stops at 2^21 since numbers are bounded by ±10^6; the maximum sum magnitude that matters is within that range.</li><li class="text-gray-300">Avoid double counting by always pairing the current element with previously seen elements only and by allowing the pair (i, j) with i == j when the same value was seen earlier (counts include prior occurrences).</li><li class="text-gray-300">Common pitfalls: iterating unnecessary powers of two beyond the sum bounds or using a set instead of counts and losing multiplicity information.</li><li class="text-gray-300">Alternative: sort the numbers and use two pointers for each power target.
